 What is this?
 --------------
 This is a application which allows you to create simple XML files which can be loaded in Unity to display
 NPC conversations.
 
 
 How to use?
 --------------
 
The first textfield is the name of the talking person.
The textarea is the text the person says.
After it press next Page to save this part of the conversations and be able to add more 


or

to fill in a FileName
then press Write to file to save it in C://TheGivenName.xml
